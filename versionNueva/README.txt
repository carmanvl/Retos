PROYECTO RETOS

Institución: Instituto de Agricultura Sostenible, CSIC Córdoba
Autor: Carlos Manuel Vinagre Lorenzo
Descripción: Aplicación web tipo dashboard interactivo dedicada a agricultores de la oliva.

Tecnologías:
- Lenguaje de programación Python
- Frameworks Flask, Dash y Bootstrap
- Repositorio Github

Estructura:
Retos
-env
-src
--assets
--pages
---ajustes.py
---anadirFinca.py
---fincas.py
---login.py
---not_found_404.py
--app.py
-README.txt
-requirements.txt

En app.py se encuentra lo relativo a la estructura general de la página, incluido el sidebar, la temática y el layout principal.
En las páginas de la carpera pages se encuentran tanto los componentes como la estructura y el layout de las correspondientes vistas.
En requirements.txt se encuentra el entorno necesario para la aplicación.
